ESTRUCTURA DE ETIQUETAS
Etiqutas de apertura Etiquetas que tienen una apertura y un sierre
Este tipo de etiqueta comúnmente se utilizan para indicar una dirección u origen se componen por argumento y opciones tales como src href Este tipo de etiquetas comúnmente se utilizan para encerrar un contenido tales como texto img entre otros
